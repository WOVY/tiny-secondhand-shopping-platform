# Tiny Second-hand Shopping Platform

WHS(화이트햇스쿨) Secure Coding 과제 — 간단한 중고거래 플랫폼.

## 상태

프로젝트 초기 세팅 중입니다. 구현 계획은 이슈 트래커를 참고하세요.

## 기술 스택 (예정)

- Node.js + Express
- EJS (서버 렌더링)
- SQLite (better-sqlite3)
- Socket.IO (실시간 채팅)

## 환경 설정 및 실행 방법

```bash
npm install
cp .env.example .env   # 필요시 SESSION_SECRET 등 값 수정
npm start               # http://localhost:5000
```

기능이 추가될 때마다 이 섹션은 계속 갱신됩니다.
